Stranded

Build/dismantle blocks to climb you way outta cave on an alien world

Controls

game requires keyboard and mouse to play
  • AD- move
  • Space - jump
  • LMB - add/remove block
  • RMB(hold) - look around
  • Escape - menu
  • 1,2,3,4 - switch block type
cursed gamepad controls:
  • D-pad - move
  • LT - jump
  • RT - add/remove block
  • Right Stick - Aim/Look
  • Right Stick press - Reset Aim/Look
  • Start - menu/pause
  • LB, RB - switch block type
